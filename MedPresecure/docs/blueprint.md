# **App Name**: MedPresecure Login

## Core Features:

- Login: Allow users to log in using their Aadhaar number and password, utilizing Firebase Authentication by converting the Aadhaar number into a dummy email format.
- Sign Up: Enable new users to create accounts with their Aadhaar number and password.
- Forgot Password: Allow users to reset their password if they have forgotten it.
- Show Password: Allow users to reveal or hide their password for easier entry and verification.
- Dual-Panel Layout: Implement a split-screen layout with a branding section on the left and a login form on the right.

## Style Guidelines:

- Primary color: Bright blue (#29ABE2) for main buttons and interactive elements, reflecting a sense of trust and security.
- Background color: Very light blue (#F0F9FF) to provide a clean, airy feel for the right panel.
- Accent color: Deep teal (#008080) for secondary elements like subtext and icons, complementing the primary blue and enhancing visual appeal.
- Font: 'Inter', a sans-serif font, for both body text and headlines, ensuring a modern and readable interface.
- Use simple, consistent icons (e.g., Feather icons) for form fields and interactive elements.
- Utilize Tailwind CSS grid and flexbox for a responsive split-screen layout.
- Subtle animations on button hover and form input focus to improve user experience.
